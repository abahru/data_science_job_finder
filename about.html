About This Project

Data Science Jobs is a web-based application built to simplify job searching in the data industry. It enables users to:

  • Search real-time job listings for roles such as Data Scientist, Data Analyst, and Machine Learning Engineer.
  • Filter by location, salary range, and job type.
  • Save interesting job posts for later.
  • Visualize salary trends and industry demand using interactive charts.

Technologies Used

  • HTML, CSS, JavaScript (Vanilla JS)
  • Chart.js for data visualization
  • Node.js & Express for backend API
  • Adzuna Job Search API for real-time listings
  • Supabase for database integration (optional future extension)
